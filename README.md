## Instrucoes

Clonar o projeto

```sh
$ git clone https://github.com/brunovmc/codinoite.git

$ cd codinoite
```
Crie sua propria branch

```
git checkout -b [nome da sua branch]
```
Para conferir em que branch esta
```
git branch
```
Para mudar de branch
```
git checkout [nome da branch]
```
